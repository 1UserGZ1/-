### 应用概述
《简易准星》是一款 Windows 桌面工具，提供可自定义的屏幕准星（矢量/图片/GIF），支持透明、鼠标穿透，仅通过系统托盘图标交互。

### 测试依赖与环境要求
- **操作系统**：Windows 10 版本 1809 或更高，Windows 11（所有版本）。
- **运行时依赖**：
  - 如果下载的是**自包含版本**，无需安装任何运行时。
  - 如果下载的是**依赖框架版本**，需要安装 [.NET 8.0 桌面运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)（选择 “运行桌面应用” 的版本）。
- **无第三方驱动或 NT 服务**：不依赖任何非 Microsoft 内核驱动或系统服务。
- **无管理员权限要求**：普通用户权限即可运行，不会请求 UAC 提升。

### 安装与首次运行
- 安装包安装后，程序会自动启动并在系统托盘显示图标（一个准星图标）。
- 首次运行默认显示**红色十字准星**覆盖全屏，鼠标可穿透点击下方窗口。

### 测试账户与访问凭证
- **不需要登录或注册**：应用完全离线运行，无任何账号系统。
- **无隐藏功能或锁定内容**：所有功能均通过设置窗口直接调节，无需解锁步骤。

### 核心功能测试步骤
#### 1. 托盘交互
- **左键双击**：默认打开设置窗口（可在通用设置中改为切换准星显示）。
- **右键单击**：显示菜单，包含“打开设置”、“显示/隐藏准星”、“退出”。
- **鼠标悬停**：显示工具提示“简易准星”。

#### 2. 设置窗口（需从托盘右键打开）
- **矢量准星标签**：
  - 启用/禁用矢量准星。
  - 切换样式（十字/圆点/T型/十字带缺口）。
  - 调整颜色（输入 HTML 颜色代码，如 #FF0000）。
  - 调节粗细、大小、X/Y 偏移（滑动条实时生效）。
- **图片准星标签**：
  - 启用/禁用图片准星。
  - 点击“选择图片”加载本地图片（支持 PNG/JPG/BMP/GIF 静态/动态）。
  - 预览图显示所选图片。
  - 调整缩放、旋转、透明度、偏移（滑动条实时生效）。
- **通用标签**：
  - 总开关：显示/隐藏所有准星。
  - 鼠标穿透：开启后点击准星区域可操作下层窗口。
  - 双击行为：选择托盘图标双击是打开设置还是切换显示。
  - “重置默认设置”按钮恢复出厂配置。

#### 3. 配置保存与验证
- 所有调节均**实时生效**，关闭设置窗口后自动保存至 `settings.json`（位于应用目录）。
- 重启应用应恢复上次的设置。

#### 4. 鼠标穿透功能验证
- 开启“鼠标穿透”后，鼠标点击准星区域应能点击到下层窗口（如浏览器、游戏）。
- 关闭后，鼠标点击准星区域无响应（但准星本身不处理点击，实际上点击被窗口拦截，测试时可在准星上放置一个透明按钮验证，但本应用无此功能，可通过观察能否点击下层判断）。

#### 5. 多显示器支持
- 主窗口默认覆盖**主屏幕**全屏（若需要多屏覆盖，需调整代码，当前仅主屏）。
- 测试时可在扩展显示器上观察准星是否只出现在主屏。

#### 6. 性能验证
- 任务管理器查看内存占用应 < 50MB，CPU 空闲时 < 5%。
- 拖动滑块调节参数时，准星刷新流畅无卡顿。

### 已知限制与注意事项
- **GIF 图片准星**：需要安装 [WpfAnimatedGif](https://github.com/XamlAnimatedGif/XamlAnimatedGif) 库支持（已内置），若用户自行替换为其他格式可能无法动画。
- **图片路径**：如果用户移动了图片文件，准星将无法显示（程序不会复制图片）。
- **全屏覆盖**：窗口始终置顶，可能覆盖某些全屏游戏或应用的界面，可通过“显示/隐藏准星”临时关闭。
- **无声音功能**：不涉及背景音频，无需测试音频相关。

### 测试完成标准
- 所有设置项均能正常调节并生效。
- 托盘图标显示正常，菜单响应正确。
- 准星在不同分辨率下位置居中，偏移调节有效。
- 鼠标穿透开关能正确切换穿透状态。
- 退出程序后托盘图标消失，无残留进程。
